# 🌍 Sistema de Gestión de Impacto Ambiental

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![SQLite](https://img.shields.io/badge/Database-SQLite3-lightgrey.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

[cite_start]Este proyecto nace como una solución integral para empresas que buscan auditar y centralizar el control de su huella ecológica[cite: 8]. [cite_start]Mediante una interfaz intuitiva, permite registrar actividades operativas, categorizarlas y cuantificar su impacto ambiental de manera sistemática[cite: 12, 13].

---

## 🚀 Funcionalidades Principales

* [cite_start]**Gestión de Datos (CRUD):** Control total sobre el ciclo de vida de los registros (Altas, Bajas, Modificaciones y Consultas)[cite: 23, 61].
* [cite_start]**Análisis Estadístico:** Cálculo automático del impacto ambiental total y promedio para identificar áreas críticas de la operación[cite: 26, 186].
* [cite_start]**Contexto en Tiempo Real:** Integración con servicios meteorológicos mediante Web Scraping para visualizar la temperatura de CABA al momento de la carga[cite: 24, 183].
* [cite_start]**Ergonomía Visual:** Soporte nativo para Modo Claro y Modo Oscuro, adaptándose a las preferencias del usuario[cite: 25, 168].

## 🛠️ Tecnologías Utilizadas

* [cite_start]**Lenguaje:** Python 3.x[cite: 3].
* **Interfaz Gráfica:** Tkinter.
* [cite_start]**Persistencia:** SQLite3[cite: 23].
* [cite_start]**Web Scraping:** BeautifulSoup & Requests[cite: 182].
* [cite_start]**Validaciones:** Expresiones Regulares (Regex)[cite: 25].

## 📋 Metodología de Medición

[cite_start]El sistema utiliza una escala de ponderación simplificada para transformar observaciones cualitativas en datos cuantitativos procesables[cite: 15, 16]:

| Valor | Clasificación | Descripción |
| :--- | :--- | :--- |
| **-1** | Impacto Negativo | [cite_start]Actividades que degradan el parámetro ambiental[cite: 17]. |
| **0** | Impacto Neutro | [cite_start]Actividades sin variaciones significativas[cite: 18]. |
| **1** | Impacto Positivo | [cite_start]Acciones que favorecen la recuperación del entorno[cite: 19]. |
